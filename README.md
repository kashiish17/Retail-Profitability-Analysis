#  Retail Profitability & Customer Insights Analysis

## Overview

This project analyzes retail sales data to evaluate profitability, customer behavior, and product performance.

##  Data Included

* Orders Data
* Customer Data
* Product Data

## Key Metrics

* Revenue
* Cost
* Profit
* Profit Margin

## Dashboard Highlights

* Total Orders
* Total Profit
* Average Order Value
* Category-wise Sales & Profit

##  Key Insights

* Electronics category shows highest profitability
* Customers aged 25–35 contribute most revenue
* High-value orders drive major profit share

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
